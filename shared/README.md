# 🌐 Shared (공유 - 혈관과 신경망)

## 📖 개요 (Overview)
이 폴더는 로봇의 모든 부품들이 공통으로 사용하는 **'공공재'**입니다. 
설정값, 통신망, 그리고 시스템의 전체 흐름을 주관하는 **혈관이자 신경망** 같은 역할을 합니다.
어느 한 레이어에 속하기 애매한, 모두에게 필요한 기능이 여기에 모여 있습니다.

---

## 📂 파일 구조 및 상세 설명 (Structure & Files)

### 1. `config.py` (설정집)
- **역할**: 시스템의 모든 **규칙과 주소**가 적힌 수첩입니다.
- **기능**:
    - "카메라는 1번 포트에 있어", "로봇 IP 주소는 192.168.0.100이야" 같은 정보가 들어있습니다.
    - 파일 경로나 AI 모델 이름도 여기서 바꿀 수 있습니다.

### 2. `pipeline.py` (중추 신경)
- **역할**: **7단계 파이프라인**을 실제로 돌리는 관리자입니다.
- **기능**:
    - 정보(데이터)가 `Sensor -> State -> Brain -> Strategy -> Expression -> Embodiment -> Memory` 순서로 흐르도록 교통 정리를 합니다.
    - 각 단계가 순서를 어기지 않게 감시합니다.

### 3. `state_broadcaster.py` (방송국)
- **역할**: 시스템 내부의 **뉴스 방송국**입니다.
- **기능**:
    - "뇌가 생각을 마쳤습니다!", "로봇이 움직이기 시작합니다!" 같은 **이벤트**를 전체 시스템에 뿌립니다.
    - 이를 구독(Subscribe)하고 있는 부품들이 소식을 듣고 자기 할 일을 합니다. (예: UI 화면이 방송을 듣고 로그를 표시함)

### 4. `intents.py` & `ui_dto.py` (공용 언어)
- **역할**: 부품끼리 대화할 때 쓰는 **'표준어 사전'**입니다.
- **기능**:
    - "명령은 이런 형식으로 보내자", "로봇 상태는 이런 글자로 표현하자"고 약속해두어, 서로 말이 통하게 합니다.
