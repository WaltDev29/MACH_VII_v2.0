# 👵 MACH_VII: Silver Physical AI Sitter Vision

**"단순한 로봇을 넘어, 어르신의 눈과 손이 되어주는 지능형 돌봄 동반자"**

## 🎯 프로젝트 목적
MACH_VII v2.0은 고령화 시대의 어르신들을 위해 **시각적 인지 보조(눈)**, **물리적 가사 및 케어 보조(손)**, 그리고 **정서적 교감(말동무)**을 수행하는 '실버 피지컬 AI 시터' 구축을 최종 목적으로 합니다.

## 🌟 주요 시나리오 (Key Use Cases)
1.  **시각 인지 보조 (Reading Assistant)**: 
    *   "이 약병에 뭐라고 써있니?" -> **Steadycam Mode**로 고해상도 캡처 후 분석하여 낭독.
    *   "내 안경 어디있니?" -> **Exploration Mode**로 집안을 탐색하여 안경 위치 파악 및 안내.
2.  **정밀 물리 케어 (Physical Care)**:
    *   "소화제 좀 가져다주렴." -> DB에 저장된 약통 위치로 이동하여 **Exploitation Mode**로 안정적 파지 및 전달.
    *   "오른쪽 등이 좀 가렵구나." -> **Safety Mode**와 실시간 `arm_status`를 감시하며 안전한 힘 조절로 등 긁어주기 수행.
3.  **성장하는 반려 에이전트 (Adaptive Companionship)**:
    *   어르신의 어투와 감정을 학습하여 **Expression Layer**를 통해 더 따뜻한 표정과 액션 생성.
    *   **FalkorDB**에 저장된 과거의 시도와 성공 경험을 바탕으로, 시간이 갈수록 어르신의 취향(예: 등이 가려운 정확한 지점)을 더 잘 알게 되는 개인화된 시터로 진화.

## 🛠️ 기술적 실현 가능성 (Technical Foundation)
*   **Hierarchical Vision**: 그리퍼 카메라(정밀 조작)와 월드 카메라(전체 경로)의 협력으로 복잡한 집안 환경 극복.
*   **3-Mode Strategy**: 어르신과의 신체 접촉 시 절대적 안전(Safety)을 보장하면서도, 새로운 장소나 물체에 적응(Exploration)하는 균형 확보.
*   **Control Tower**: 로봇이 본인의 상태(끼임, 흔들림 등)를 실시간으로 인지하여 LLM이 스스로 계획을 수정하는 지능형 루프 구축.
